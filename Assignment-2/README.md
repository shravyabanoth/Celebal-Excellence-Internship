# Week 2 Assignment - End-to-End ML Pipeline

This project implements a complete Machine Learning pipeline on Tesla sales and production data.

## Topics Covered
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Encoding Techniques
- Feature Engineering
- Linear Regression
- Random Forest Regression
- Model Evaluation
- Cross Validation
- Hyperparameter Tuning
- Time Series Forecasting using ARIMA

## Dataset
Tesla EA Deliveries and Production Data (2015–2025)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Statsmodels
