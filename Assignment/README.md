## Objective
The objective of this assignment is to build and compare three deep learning models—SimpleRNN, LSTM, and GRU—for text generation. The models are trained on a custom text corpus to learn sequential patterns and generate meaningful text from a given seed phrase.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Project Workflow
1. Import required libraries
2. Prepare a custom text corpus
3. Tokenize the text
4. Generate n-gram input sequences
5. Apply sequence padding
6. Prepare training data
7. Train the SimpleRNN model
8. Train the LSTM model
9. Train the GRU model
10. Compare training loss using a line graph
11. Generate text using all three models
12. Compare the performance of the models

## Results
- Successfully trained SimpleRNN, LSTM, and GRU models.
- Compared the training loss of all three models over 100 epochs.
- Generated text from the same seed phrase using each model.
- Observed that all models learned the sequence patterns effectively on the custom dataset.
