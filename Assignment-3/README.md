# Week 3 Assignment - Customer Intelligence System

This project applies clustering and classification techniques on the Bank Marketing dataset to analyze customer behavior and predict term deposit subscriptions.

## Topics Covered

* Data Cleaning
* Feature Encoding
* Feature Scaling
* K-Means Clustering
* DBSCAN Clustering
* PCA Visualization
* Cluster Analysis
* Random Forest Classification
* XGBoost Classification
* Model Comparison

## Dataset

Bank Marketing Dataset
