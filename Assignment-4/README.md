# Assignment 4 – Image Classification using ANN and CNN

## Objective

The objective of this assignment is to build image classification models using Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN) on the CIFAR-10 dataset and compare their performance across different architectures and training strategies.

## Dataset

* Dataset: CIFAR-10
* Training Images: 50,000
* Testing Images: 10,000
* Classes: 10
* Image Size: 32 × 32 RGB

## Tasks Performed

* Data Loading and Preprocessing
* Data Normalization
* One-Hot Encoding
* Data Visualization
* ANN Model with Dropout
* CNN Model with Batch Normalization
* Validation Accuracy Comparison
* Advanced CNN with Data Augmentation
* Early Stopping Implementation
* Model Performance Analysis

## Results

| Model        | Test Accuracy |
| ------------ | ------------- |
| ANN          | 40.21%        |
| Basic CNN    | 65.04%        |
| Advanced CNN | 64.10%        |

## Conclusion

The CNN-based models significantly outperformed the ANN model for image classification tasks. The experiment demonstrates the effectiveness of convolutional neural networks in extracting meaningful visual features from image data.
